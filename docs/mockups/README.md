# UI Mockups

Save the generated mockup images from the Amp thread to this folder with the following names:

1. `01-mobile-weekly-overview.png` - Mobile weekly grid view
2. `02-mobile-daily-focus.png` - Mobile daily task list
3. `03-mobile-notes.png` - Mobile notes & narrations
4. `04-ipad-duet-view.png` - iPad split view (weekly grid + daily focus)
